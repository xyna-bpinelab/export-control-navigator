## 効果的な参照方法

### 該非判定を行う場合

1. [輸出令別表第１（e-Gov）](https://laws.e-gov.go.jp/law/325CO0000000378)または[外国為替令別表（e-Gov）](https://laws.e-gov.go.jp/law/355CO0000000260)で対象品目の項番を確認
2. METIサイトから該当項目の**パラメーターシート**をダウンロードして技術的基準と照合
3. 非該当の場合も[キャッチオール規制ページ](https://www.meti.go.jp/policy/anpo/catch_all.html)で適用可否を確認

### 許可申請を行う場合

1. [許可申請ページ](https://www.meti.go.jp/policy/anpo/apply.html)で申請様式をダウンロード
2. [NACCS](https://www.naccscenter.com/)経由での電子申請も可能
3. 不明点は[安全保障貿易審査課](https://www.meti.go.jp/policy/anpo/contact.html)に事前相談することを推奨

### 内部規程（ICP）を整備する場合

1. [ICPページ](https://www.meti.go.jp/policy/anpo/jisyukanri.html)でMETI公表のICP参考例・チェックリストを取得
2. [説明会資料](https://www.meti.go.jp/policy/anpo/seminer/index.html)の最新版でポイントを確認

## 参照時の注意ポイント

{{< alert color="info" >}}
**最新情報の確認**：法令改正・通達改正は予告なく行われることがあります。トップページの新着情報を定期的に確認し、改正内容を把握してください。
{{< /alert >}}

{{< alert color="info" >}}
**一次情報の確認**：本サイトを含む解説サイトは参考情報です。許可要否の最終判断は、METIの法令・通達等の一次情報に基づいて行ってください。
{{< /alert >}}

{{< alert color="warning" >}}
**グループ分類の変更**：輸出先の国・地域のグループ分類（グループA〜D）は政治情勢等により変更されることがあります。[輸出令別表第３（e-Gov）](https://laws.e-gov.go.jp/law/325CO0000000378)の最新版を必ず確認してください。
{{< /alert >}}
