経済産業省（METI）の安全保障貿易管理ページ（[https://www.meti.go.jp/policy/anpo/index.html](https://www.meti.go.jp/policy/anpo/index.html)）は、日本の輸出管理制度に関する一次情報源です。

## サイトの主要構成

METIの安全保障貿易管理ページは、大きく以下のカテゴリに分類されています。

### 1. 法令・通達

輸出管理の根拠となる法令・政省令・告示・通達が掲載されています。

- **外国為替及び外国貿易法（外為法）**
- **輸出貿易管理令（輸出令）**
- **外国為替令**
- 関連告示・通達（経済産業省令、パラメーターシートなど）

{{< alert color="warning" >}}
法令・通達は頻繁に改正されます。e-Gov法令検索や官報と合わせて最新版を確認してください。
{{< /alert >}}

### 2. 許可申請手続き

輸出許可の申請方法・様式が掲載されています。

- 個別輸出許可申請
- 包括輸出許可申請（一般包括・特別一般包括・特定輸出者）
- 役務取引許可申請

### 3. 該非判定・用途確認

自社貨物・技術がリスト規制に該当するかどうかを判断するための資料が掲載されています。

- **パラメーターシート**（規制品目の技術的パラメーター一覧）
- **マトリクス**（需要者リストの確認）
- 該非判定ガイダンス

### 4. セミナー・相談窓口

- 安全保障貿易管理説明会の資料・動画
- 安全保障貿易審査課への相談窓口案内

## 効果的な参照方法

### 該非判定を行う場合

1. **輸出令別表第１**または**外国為替令別表**で対象品目の項番を確認
2. 該当項目の**パラメーターシート**をダウンロードして技術的基準と照合
3. 非該当の場合も**キャッチオール規制**の適用可否を確認

### 許可申請を行う場合

1. METIサイトの「許可申請」ページで申請様式をダウンロード
2. **NACCS（輸出入・港湾関連情報処理センター）**経由での電子申請も可能
3. 不明点は**安全保障貿易審査課**に事前相談することを推奨

### 内部規程（ICP）を整備する場合

1. 「安全保障貿易管理内部規程（ICP）」のページを参照
2. METIが公表しているICP参考例・チェックリストを活用

## 参照時の注意ポイント

{{< alert color="info" >}}
**最新情報の確認**：法令改正・通達改正は予告なく行われることがあります。「新着情報」セクションを定期的に確認し、改正内容を把握してください。
{{< /alert >}}

{{< alert color="info" >}}
**一次情報の確認**：本サイトを含む解説サイトは参考情報です。許可要否の最終判断は、METIの法令・通達等の一次情報に基づいて行ってください。
{{< /alert >}}

{{< alert color="warning" >}}
**グループ分類の変更**：輸出先の国・地域のグループ分類（グループA〜D）は政治情勢等により変更されることがあります。輸出令別表第３の最新版を必ず確認してください。
{{< /alert >}}
