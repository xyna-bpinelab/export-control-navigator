経済産業省（METI）の[安全保障貿易管理ページ](https://www.meti.go.jp/policy/anpo/)は、日本の輸出管理制度に関する一次情報源です。

{{< alert color="warning" >}}
METIサイトは構成変更が行われることがあり、サブページのURLが変わる場合があります。リンク切れが発生した場合は、[トップページ](https://www.meti.go.jp/policy/anpo/)のメニューから目的のページを探してください。
{{< /alert >}}

---

## サイトの主要構成

METIの安全保障貿易管理ページは、大きく以下のカテゴリに分類されています。

### 1. 規制貨物・規制技術（マトリクス）

輸出規制の対象となる貨物・技術の一覧、パラメーターシート、需要者リストが集約されています。

- **URL**: [https://www.meti.go.jp/policy/anpo/matrix_intro.html](https://www.meti.go.jp/policy/anpo/matrix_intro.html)
- パラメーターシート（規制品目の技術的基準）のダウンロード
- 需要者リスト（ユーザーリスト）の確認
- 該非判定の参考資料

{{< alert color="info" >}}
**該非判定の起点**：自社製品・技術が規制対象かどうかを調べる際は、まずこのページから該当項番のパラメーターシートをダウンロードしてください。
{{< /alert >}}

### 2. 法令・通達

輸出管理の根拠となる法令・政省令・告示・通達が掲載されています。METIサイトの法令ページのほか、以下のe-Govリンクから常に最新の条文を参照できます。

| 法令名 | e-Govリンク |
|--------|------------|
| 外国為替及び外国貿易法（外為法） | [e-Gov](https://laws.e-gov.go.jp/law/324AC0000000228) |
| 輸出貿易管理令（輸出令） | [e-Gov](https://laws.e-gov.go.jp/law/325CO0000000378) |
| 外国為替令 | [e-Gov](https://laws.e-gov.go.jp/law/355CO0000000260) |

{{< alert color="warning" >}}
法令・通達は頻繁に改正されます。[e-Gov法令検索](https://laws.e-gov.go.jp/)で必ず最新版を確認してください。
{{< /alert >}}

### 3. 許可申請手続き

輸出許可・役務取引許可の申請方法・様式が掲載されています。個別申請・包括申請の様式はMETIトップページのメニューから「許可申請」を選択してください。電子申請は[NACCS（輸出入・港湾関連情報処理センター）](https://www.naccscenter.com/)経由で行えます。

### 4. セミナー・相談窓口

METIは定期的に安全保障貿易管理説明会を開催しており、資料・動画はMETIサイトで公開されています。制度の運用に関する個別相談は、経済産業省 貿易経済協力局 安全保障貿易審査課が窓口です。

## 効果的な参照方法

### 該非判定を行う場合

1. [規制貨物・規制技術ページ](https://www.meti.go.jp/policy/anpo/matrix_intro.html)で対象品目の項番を確認
2. 該当項目の**パラメーターシート**をダウンロードして技術的基準と照合
3. 非該当の場合も**キャッチオール規制**の適用可否を確認（同ページ内に資料あり）

### 許可申請を行う場合

1. METIトップページのメニューから「許可申請」を選択し、申請様式をダウンロード
2. [NACCS](https://www.naccscenter.com/)経由での電子申請も可能
3. 不明点は経済産業省 安全保障貿易審査課に事前相談することを推奨

### 内部規程（ICP）を整備する場合

1. METIトップページのメニューから「自主管理」を選択
2. METIが公表しているICP参考例・チェックリストを活用

## 参照時の注意ポイント

{{< alert color="info" >}}
**最新情報の確認**：法令改正・通達改正は予告なく行われます。[トップページ](https://www.meti.go.jp/policy/anpo/)の新着情報を定期的に確認してください。
{{< /alert >}}

{{< alert color="info" >}}
**一次情報の確認**：本サイトを含む解説サイトは参考情報です。許可要否の最終判断は、METIの法令・通達等の一次情報に基づいて行ってください。
{{< /alert >}}

{{< alert color="warning" >}}
**グループ分類の変更**：輸出先の国・地域のグループ分類（グループA〜D）は政治情勢等により変更されることがあります。[輸出令（e-Gov）](https://laws.e-gov.go.jp/law/325CO0000000378)の別表第３の最新版を必ず確認してください。
{{< /alert >}}
