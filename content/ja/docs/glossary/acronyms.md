輸出管理でよく使われる英語略称・略語の一覧です。

| 略称 | 正式名称 | 日本語 |
|------|----------|--------|
| AG | Australia Group | オーストラリア・グループ（化学・生物兵器関連の輸出管理レジーム） |
| CISTEC | Center for Information on Security Trade Controls | 一般財団法人 安全保障貿易情報センター |
| EUC | End User Certificate | エンドユーザー証明書 |
| FEFTA | Foreign Exchange and Foreign Trade Act | 外国為替及び外国貿易法（外為法） |
| ICP | Internal Compliance Program | 安全保障貿易管理内部規程（内部コンプライアンスプログラム） |
| METI | Ministry of Economy, Trade and Industry | 経済産業省 |
| MTCR | Missile Technology Control Regime | ミサイル技術管理レジーム |
| NACCS | Nippon Automated Cargo and Port Consolidated System | 輸出入・港湾関連情報処理センター（電子申請システム） |
| NSG | Nuclear Suppliers Group | 核供給国グループ |
| WA | Wassenaar Arrangement | ワッセナー・アレンジメント（通常兵器・汎用品技術の輸出管理レジーム） |
| WMD | Weapons of Mass Destruction | 大量破壊兵器 |
