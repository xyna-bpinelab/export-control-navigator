[CISTEC（一般財団法人 安全保障貿易情報センター）](https://www.cistec.or.jp/)は、輸出管理の民間支援機関として、情報提供・相談・研修・認定試験など幅広いサービスを提供しています。METIサイトと並ぶ重要な情報源です。

---

## サイトの主要構成

| セクション | URL | 内容 |
|-----------|-----|------|
| 輸出管理基本情報 | [/export/index.html](https://www.cistec.or.jp/export/index.html) | 制度解説・法令・改正情報・違反事例等 |
| 書籍・出版物・映像 | [/publication/index.html](https://www.cistec.or.jp/publication/index.html) | テキスト・問題集・ガイドブック等の購入 |
| 一般サービス | [/service/index.html](https://www.cistec.or.jp/service/index.html) | 相談・セミナー・eラーニング等 |
| 実務能力認定試験 | [/nintei/index.html](https://www.cistec.or.jp/nintei/index.html) | STC認定試験の情報 |
| 公表リスト | [/kohyo/index.html](https://www.cistec.or.jp/kohyo/index.html) | ICリストなし品目の確認 |

## 1. 輸出管理基本情報

**URL**: [https://www.cistec.or.jp/export/index.html](https://www.cistec.or.jp/export/index.html)

輸出管理の制度・法令に関する解説コンテンツが集約されています。

| コンテンツ | 内容 |
|-----------|------|
| 輸出管理の基礎 | 制度の仕組みをわかりやすく解説 |
| 関係法規 | 外為法・輸出令・外国為替令等の解説 |
| 政省令等改正 | 法令改正の最新情報・改正経緯 |
| 経済産業省通達 | METIが発出した通達の解説 |
| 許可申請手続 | 各種許可申請の手順 |
| 自主管理 | ICP（内部規程）の整備指針 |
| 外為法違反事例 | 実際の違反事例の概要 |
| 経済制裁措置 | 各国制裁制度の解説 |
| 防衛装備移転三原則 | 武器輸出ルールの解説 |
| 輸出管理用語集 | 専門用語の解説集 |

{{< alert color="info" >}}
**活用ポイント**：法令改正が行われた際には「政省令等改正」のページで改正概要と施行日を素早く確認できます。METIの一次情報と合わせて参照してください。
{{< /alert >}}

## 2. 公表リスト検索システム

**URL**: [https://ap.cistec.or.jp/klk/](https://ap.cistec.or.jp/klk/)

ICメーカー約29社、約13万型番について、CISTECがリスト非該当と判断した品目を検索できます。

- 型番（3文字以上）で検索可能、24時間利用可能
- 対象型番がヒットした場合、インボイスにメーカー名・型番・公表日を記載することで、メーカーからの該非判定書なしに輸出手続きが可能

{{< alert color="info" >}}
**活用ポイント**：IC部品の調達・輸出が多い企業では、まずこの検索システムで確認するとスクリーニング作業を効率化できます。
{{< /alert >}}

## 3. 輸出管理相談サービス

**URL**: [https://www.cistec.or.jp/service/sodan/zizensodan.html](https://www.cistec.or.jp/service/sodan/zizensodan.html)

外為法第25条・第48条に関する具体的な相談をCISTECの専門家に行えます。

### 料金

| 区分 | 料金（税込） |
|------|------------|
| 賛助会員（企業） | 無料 |
| 一般企業 | 初回20分 5,500円、以降20分ごと 3,300円 |
| 大学会員 | 年間15回まで無料、16回目以降 3,300円 |
| その他大学 | 年間15回まで 8,800円、16回目以降 3,300円 |

### 注意事項

- 予約制（3週間先まで）、Webまたは対面のみ（電話相談不可）
- 1回あたり最長60分
- 個人的な相談・架空の相談は対象外

{{< alert color="warning" >}}
METIへの事前相談と合わせて活用することで、許可申請前の不確実性を低減できます。
{{< /alert >}}

## 6. メール配信サービス（無料）

**URL**: [https://www.cistec.or.jp/service/hotline/index.html](https://www.cistec.or.jp/service/hotline/index.html)

法令改正・セミナー情報等をメールで受け取れます。いずれも**無料**で登録可能です。

| サービス名 | 配信内容 |
|-----------|---------|
| CISTEC Hotline | データベース更新情報・新刊・政省令改正説明会等 |
| CISTEC インフォ・メール（IM） | 新刊・認定試験・セミナー等のご案内 |

{{< alert color="info" >}}
**活用ポイント**：法令改正の情報をタイムリーに把握するために、どちらか一方への登録を強くお勧めします。フリーメールアドレスは登録不可です。
{{< /alert >}}

## 賛助会員限定サービス

以下のデータベースは賛助会員向けのサービスです。

| サービス | 内容 |
|---------|------|
| 国内法令コーナー | 輸出管理関連法令の全文・改正履歴 |
| CHASERコーナー | 外国法令・各国制度情報 |
| 該非判定コーナー | 品目別の詳細な判定支援情報 |
| ガイダンスコーナー | 実務手続きの詳細解説 |
| 統合制裁リスト | 各国制裁対象者リストの統合検索 |

{{< alert color="info" >}}
輸出管理業務の頻度が高い企業は、賛助会員への加入によりこれらのデータベースを活用することで業務効率が大幅に向上します。
{{< /alert >}}
