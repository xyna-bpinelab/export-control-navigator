日本の輸出管理は、**外為法**を頂点とする法令体系に基づいています。
法律→政令→省令・告示→通達の順に、より具体的な規制内容が規定されます。

```
外国為替及び外国貿易法（外為法）  ←  国会が制定する「法律」
        │
        ├── 輸出貿易管理令（輸出令）  ←  内閣が制定する「政令」（貨物）
        │         └── 輸出貿易管理規則  ←  経産省が制定する「省令」
        │                   └── 関連告示・通達
        │
        └── 外国為替令              ←  内閣が制定する「政令」（技術）
                  └── 関連告示・通達
```
