## 概要

**輸出貿易管理令**（輸出令）は、外為法第48条に基づき、輸出規制の対象となる貨物の品目・地域・許可手続き等を定めた政令です。

---

## 主要な別表（附属書）

### 別表第１ — 規制品目リスト（リスト規制）

輸出許可が必要な貨物を国際輸出管理レジームの分類に対応する形で列挙しています。

| 項番 | 分類名 | 対応レジーム |
|------|--------|-------------|
| 1項 | 武器 | WA（軍需品リスト） |
| 2項 | 核兵器関連 | NSG（トリガーリスト） |
| 3項 | 化学兵器関連 | AG |
| 4項 | 生物兵器関連 | AG |
| 5項 | ミサイル関連 | MTCR |
| 6項 | 先端材料 | WA（デュアルユース） |
| 7項 | 材料加工 | WA（デュアルユース） |
| 8項 | エレクトロニクス | WA（デュアルユース） |
| 9項 | コンピュータ | WA（デュアルユース） |
| 10項 | 通信・情報セキュリティ | WA（デュアルユース） |
| 11項 | センサ・レーザ | WA（デュアルユース） |
| 12項 | 航法・航空電子 | WA（デュアルユース） |
| 13項 | 海洋 | WA（デュアルユース） |
| 14項 | 推進装置 | WA（デュアルユース） |
| 15項 | その他（NSG汎用品等） | NSG（汎用品リスト）等 |
| 16項 | 炭素繊維等 | — |

{{< alert color="info" >}}
**1項（武器）**は原則として輸出不可（防衛装備移転三原則に基づく）。2〜16項が「リスト規制」の主な対象。
{{< /alert >}}

### 別表第２ — 非規制品目

輸出令の規制対象外となる品目（食品・一般消費財等）を規定。

### 別表第３（グループA）— ホワイト国

キャッチオール規制（通常兵器）の適用が除外される国・地域のリスト。輸出管理体制が整備された友好国。

{{< alert color="warning" >}}
グループ分類は政令改正により変更されます。2023年に韓国がグループAに再指定されるなど、国際情勢に応じて見直されます。必ず最新の政令を確認してください。
{{< /alert >}}

### 別表第３の２（グループB）

大量破壊兵器キャッチオール規制の「客観要件」における地域分類に使用。

### 別表第３の３（グループC）・別表第３の４（グループD）

通常兵器キャッチオール規制等に関連する地域分類。

## 許可の種類（輸出令第4条等）

| 許可の種類 | 概要 |
|------------|------|
| 個別輸出許可 | 輸出の都度申請する基本的な許可 |
| 一般包括輸出許可 | 特定地域・用途・需要者を対象とした包括許可 |
| 特別一般包括輸出許可 | ICPの整備を前提とした広範な包括許可 |
| 特定輸出許可 | 特定輸出者（大臣認定）に付与される包括許可 |
