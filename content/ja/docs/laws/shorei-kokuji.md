## 法令体系における位置づけ

省令・告示・通達は、法律・政令が定める枠組みの中で、実務上の手続きや技術的基準の詳細を規定します。

```
法律（外為法）
  └── 政令（輸出令・外国為替令）
        └── 省令（輸出貿易管理規則 等）
              └── 告示（パラメーター告示 等）
                    └── 通達（運用指針 等）
```

---

## 省令

### 輸出貿易管理規則

輸出令の委任を受け、輸出許可申請の手続き・様式・添付書類等を定めます。

- 許可申請書の様式
- 包括許可の申請要件・管理体制の基準
- 特定輸出者の認定手続き

---

## 告示

### パラメーター告示（規制品目の技術的基準）

輸出令別表第１・外国為替令別表の各項に対応する形で、規制の対象となる具体的な技術的数値・性能基準（パラメーター）を定めます。

{{< alert color="info" >}}
**実務上の重要性**：自社製品・技術の該非判定において中心的な役割を果たします。METIサイトからダウンロードできる「**パラメーターシート**」は、この告示の内容を一覧表形式にまとめたものです。
{{< /alert >}}

### 地域告示

輸出令別表第３（グループA）等の地域分類の内容を具体化する告示。輸出先国・地域の分類を確認する際に参照します。

---

## 通達

### 輸出管理に関する通達の種類

通達は法的拘束力を持たない行政内部の指示ですが、経済産業省の解釈・運用方針を示すものとして実務上重要な意味を持ちます。

| 通達の種類 | 内容 |
|------------|------|
| キャッチオール規制関連通達 | 大量破壊兵器・通常兵器キャッチオール規制の運用指針 |
| 包括許可関連通達 | 一般包括・特別一般包括許可の申請要件・管理体制の解説 |
| ICP関連通達 | 安全保障貿易管理内部規程（ICP）の整備指針 |
| みなし輸出関連通達 | 国内における技術提供規制の運用指針 |
| 需要者リスト関連通達 | ユーザーリストの掲載基準・取扱いに関する指針 |

{{< alert color="warning" >}}
**通達の確認方法**：METIの安全保障貿易管理ページの「法令・通達」セクションから最新の通達を確認してください。通達は告示・法令改正に合わせて随時改正されます。
{{< /alert >}}

## まとめ：実務での参照順序

| 目的 | 参照する法令等 |
|------|---------------|
| 規制対象かどうかの確認（貨物） | 輸出令別表第１ → パラメーター告示 → パラメーターシート |
| 規制対象かどうかの確認（技術） | 外国為替令別表 → パラメーター告示 |
| キャッチオール規制の適用確認 | 輸出令 → 地域告示（グループ分類）→ キャッチオール通達 |
| 許可申請手続きの確認 | 輸出貿易管理規則 → METIサイトの申請様式 |
| 包括許可・ICP要件の確認 | 省令 → 包括許可通達・ICP通達 |
